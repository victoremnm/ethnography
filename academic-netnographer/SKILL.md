# Academic Netnography Skill

## Core Workflow: The "Refute/Contest" Flywheel

1. **Ingest:** Use `google_web_search` and `read_file` to gather community narratives and primary sources.
2. **Fact-Check:** Conduct on-chain forensics (Dune/Axiom) or infrastructure audits to identify "Success vs. Reality" gaps.
3. **Cite:** Document findings using APA 7th Edition standards for digital/AI sources.
4. **Contest:** Explicitly seek out the "Devil's Advocate" position from Tier 1 skeptics.

## APA 7th Edition Citation Guide (2026 Standards)

### X (Twitter) Threads
- **Format:** Author, A. A. [@username]. (Year, Month Day). *First 20 words...* [X thread]. X. URL
- **Example:** Jezos, B. [@BasedBeffJezos]. (2026, February 18). *Even the magma yearns for form. The thermodynamic god...* [X thread]. X. https://x.com/beffjezos/123

### Farcaster Casts
- **Format:** Author, A. A. [@username]. (Year, Month Day). *First 20 words...* [Cast]. Warpcast. URL
- **Example:** Buterin, V. [@vitalik.eth]. (2026, January 16). *2026 is the year we take back lost ground in computing self-sovereignty...* [Cast]. Warpcast. https://warpcast.com/vitalik.eth/0xabc

### AI Agent / Large Language Model Output
- **Format:** Company. (2026). *Name of AI* (Version) [Large language model]. URL
- **Example:** Google. (2026). *Gemini* (March 1 version) [Large language model]. https://gemini.google.com/

### Technical Documentation / Software
- **Format:** Author/Company. (Year, Month Day). *Title of Section*. Website Name. URL
- **Example:** Extropic. (2026, February). *Thermodynamic Computing Roadmap: Z1 Chip*. https://extropic.ai/Z1

## Adversarial Research Framework

Every research deep-dive MUST include a **"Contested Views"** section challenging the community's core assumptions:
- **The "API Tether" Problem:** Is the community truly sovereign or just wrapping centralized Big Tech?
- **The "Extraction Machine" Lens:** Is the value being created (positive-sum) or just extracted (negative-sum)?
- **The "Luxury Belief" Risk:** Is the ideology only viable for the digital elite?
