# Netnography Principles (Kozinets, 2020)

## Core Methods

1. **Naturalistic Observation:**
   - Observe digital communities in their "natural habitat" (Farcaster, X, closed Discords).
   - Avoid intrusive questioning; prioritize the analysis of existing social artifacts.

2. **Trace Data Ingestion:**
   - Leverage the permanent digital record: on-chain transactions, archived posts, GitHub commit history.
   - Use automated tools (Dune, Kaito) to map the "Reality" behind the "Narrative."

3. **Participant Observation:**
   - The researcher must "live" in the community to understand shibboleths (e.g., *Magma Koan*, *Agentic GDP*).
   - Maintain a "Reflexive Journal" to acknowledge personal biases.

4. **Iterative Coding:**
   - Use "In Vivo" coding: using the community's own language to categorize behavior.
   - Move from raw data → categories → theological/cultural themes.

## Ethical Standards
- **Informed Consent:** Required for private digital spaces.
- **Public Figure Exception:** Analysis of high-profile leaders (Vitalik, Beff Jezos) is considered public discourse study.
- **Anonymization:** Always anonymize retail/non-public participants to prevent "doxxing."
